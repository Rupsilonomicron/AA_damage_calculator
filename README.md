# AA ダメージ計算ツール

**► https://rupsilonomicron.github.io/AA_damage_calculator/**

武器のDPS・ダメージを計算するツールです。上のURLをブックマークすれば、PC・スマホどちらでも常に最新版が使えます（アプリの更新作業は不要です）。

## 旧ローカルファイル版からのデータ移行

旧HTMLファイルで保存した武器データ・ロードアウトを引き継ぐには：

1. 旧ファイルを開き、⚙（設定）→「データ管理」→「共有コード」→「コードを生成」→ コピー
2. 上記URLのWeb版を開き、同じ画面の「インポート」にコードを貼り付けて実行
